所需器材:
	探索者STM32F4开发板
	
代码目的:
	利用ATK-SIM900/SIM900A GSM模块的GPRS功能发送采集到的温湿度以及三相电压电流的数据，送至云平台，公网IP地址为47.105.34.95
	
硬件资源:
	1,DS0(连接在PF9) 
	2,串口1(波特率:115200,PA9/PA10连接在板载USB转串口芯片CH340上面)
	3,ALIENTEK 2.8/3.5/4.3/7寸TFTLCD模块(通过FSMC驱动,FSMC_NE4接LCD片选/A6接RS) 	
	4,按键KEY0(PE4)/KEY1(PE3)/KEY2(PE2)/KEY_UP(PA0,也称之为WK_UP)
	5,SD卡,通过SDIO(SDIO_D0~D4(PC8~PC11),SDIO_SCK(PC12),SDIO_CMD(PD2))连接 
	6,W25Q128(SPI FLASH芯片,连接在SPI1上) 
	7,24C02
	8,ATK-SIM900A/SIM900 GSM模块1个,杜邦线3根/直连RS232线一条(一公一母).
	
	ATK-SIM900A模块与开发板连接方式1（通过3根杜邦线连接）：
	ATK-SIM900A          STM32开发板
		   STXD<------------>PB11(RX)
		   SRXD<------------>PB10(TX)
		   GND<------------->GND 
			
	ATK-SIM900A模块与开发板连接方式2（通过RS232串口线连接）：
	ATK-SIM900A      	STM32开发板

	9，三相电压电流变换器与开发板连接方式,（注：电压电流变换器需要24v电压）
		   电压变送器A+<------------>PA5(三相电压1)
		   电压变送器B+<------------>PA6(三相电压2)
		   电压变送器C+<------------>PA7(三相电压3) 
		  
		   电流变送器A+<------------>PA4(三相电流1)
		   电流变送器B+<------------>PC14(三相电流2)
		   电流变送器C+<------------>PC14(三相电流3) 



实验现象:	
	本实验功能简介：本设备通过GSM模块的GPRS功能发送采集到的温湿度以及三相电压电流的数据，送至云平台
